# Changelog

## 0.1.0

- Primeira versão do Seiden FLOW.
- Ingestão canônica por API.
- Assinatura de eventos do Seiden Bridge pelo barramento do Home Assistant.
- Normalização e deduplicação por `event_id`.
- Persistência SQLite com histórico imutável.
- Estado atual de pessoas e fontes.
- Dashboard por Ingress.
- API de consulta e resumo.
- Exportação JSON e CSV.
- Sensores resumidos no Home Assistant.
- Retenção automática configurável.
